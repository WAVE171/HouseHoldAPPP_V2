## Getting Started

### Creating an Account

1. Navigate to the application URL
2. Click **"Register"** on the login page
3. Fill in your details:
   - Email address
   - Password (minimum 8 characters)
   - First and Last name
   - Household name (creates a new household)
4. Click **"Create Account"**
5. You'll be automatically logged in as the household Admin

### Joining an Existing Household

If someone invites you to their household:
1. Check your email for the invitation link
2. Click the link to accept the invitation
3. Create your account with your personal details
4. You'll be added to the household with the assigned role

### User Roles

| Role | Description | Capabilities |
|------|-------------|--------------|
| **Admin** | Household administrator | Full control over all features and members |
| **Parent** | Guardian/Parent | Can manage members, view finances, invite users |
| **Member** | Regular member | Can use most features, limited admin access |
| **Staff** | Household employee | Limited access to relevant features |

## Tasks Management

### Creating a Task

1. Click **"+ New Task"** button
2. Fill in task details:
   - **Title** (required): Brief description
   - **Description**: Detailed information
   - **Priority**: Low, Medium, High
   - **Due Date**: When it should be completed
   - **Assignee**: Who should do it
   - **Tags**: Categories for organization
3. Click **"Create Task"**

### Task Views

**List View**
- Traditional list with all task details
- Sort by due date, priority, or status
- Filter by status, priority, assignee

**Kanban Board**
- Visual board with columns: To Do, In Progress, Done
- Drag and drop to change status
- Great for visual task tracking

**Calendar View**
- Tasks displayed on calendar by due date
- See workload distribution over time
- Click dates to see tasks due

### Managing Tasks

**Update Status**
- Click the status badge to cycle through states
- Or drag to different columns in Kanban view

**Edit Task**
- Click the task or use the menu (⋮)
- Select "Edit" to modify details

**Delete Task**
- Use the menu (⋮) and select "Delete"
- Confirm deletion (cannot be undone)

### Task Filters

- **Search**: Find tasks by title or description
- **Status**: Filter by Pending, In Progress, Completed
- **Priority**: Filter by Low, Medium, High
- **Assignee**: Show tasks for specific person

## Receipt Scanning

### Scanning a Receipt

1. Click **"+ Scan Receipt"**
2. Upload receipt image (photo or file)
3. System automatically extracts:
   - Store name
   - Date
   - Items and prices
   - Total amount
4. Review and edit extracted data
5. Save

### Using Scanned Data

**Create Transaction**
- Auto-fill transaction from receipt
- Categorize expenses
- Attach receipt image

**Add to Inventory**
- Add purchased items to inventory
- Set quantities
- Track purchase prices

**Link to Existing Transaction**
- Match receipt to manual transaction
- Keep records organized

### Barcode Lookup

1. Click **"Scan Barcode"**
2. Use camera to scan product barcode
3. View product information
4. Add to inventory or shopping list

## Troubleshooting

### Common Issues

**Can't log in**
1. Check email and password are correct
2. Click "Forgot Password" to reset
3. Check if account is locked (contact admin)

**Data not saving**
1. Check internet connection
2. Refresh the page
3. Try logging out and back in
4. Clear browser cache

**Missing features**
- Some features are role-restricted
- Ask your household admin for access
- Check if feature is available for your plan

**Household suspended**
- Contact support for assistance
- Data is read-only until resolved
- Check with your household admin

### Getting Help

- **In-app help**: Click the ? icon
- **Support email**: support@householdhero.com
- **Documentation**: Check docs folder
- **FAQ**: Visit help center

### Reporting Bugs

If you find a bug:
1. Note what you were doing
2. Screenshot any error messages
3. Note your browser and device
4. Report via Settings > Help > Report Bug
